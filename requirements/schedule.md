Multi-Select Grid Interactions
- **Click cell**: Toggle individual time slot
- **Click row header** (e.g., "6am"): Select/deselect all days at that time
- **Click column header** (e.g., "MON"): Select/deselect all times on that day
- **Keyboard**: Arrow keys navigate, Space toggles

Implementation Strategy
1. Frontend collects all form data
2. Generates CSV in browser using collected data
```csv
   class_type,instructor,room,date,time,capacity,duration
   Reformer,Farah,Studio A,2024-01-15,04:00,12,50
   Reformer,Farah,Studio A,2024-01-21,06:00,12,50
   Reformer,Farah,Studio A,2024-01-22,04:00,12,50
```
3. Sends to existing CSV upload endpoint
Endpoint: `POST /admin/schedules/upload`
Request: multipart/form-data with generated CSV




### Business Rules
- SuperAdmin only
- Week starts on Sunday
- Classes displayed in time order (earliest first)
- Classes for today highlighted with different background
- **Time Conversion**: Local time (UTC+3) converted to UTC when submitting
   - Form shows local time (e.g., 07:00)
   - Subtracts 3 hours for UTC (sends 04:00)
- After class creation, calendar auto-refreshes to show new classes
